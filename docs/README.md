# Documentation

Reference materials and schema notes.

## What goes here

| File | Purpose |
|------|---------|
| `data_dictionary.md` | Auto-generated by `make snapshot`. Columns, types, samples per data file. |
| Schema notes, ERDs, glossaries | Anything explaining the data, not the project. |

## What does **not** go here

- Data files → `data/`
- Final deliverables (PDFs, slides) → `reports/`
- Stakeholder one-pager → `README.md` (root)
- AI prompts → `SKILL.md` (root) or `.claude/skills/`
